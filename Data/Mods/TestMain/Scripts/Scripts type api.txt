	BuyCat - Класс кота в магазине
create_buy_cat - Создание класса
draw_buy_cat - Отрисовка класса
move_buy_cat - Движение класса при нажатие на кнопки
info_buy_cat - Отрисовка информации о классе
buy_cat - Покупка кота при нажатие на класс

	Cat - Кот на поле боя
create_cat - Создание класса
update_cat - Обновление класса
bump_cat - Обработка дамага от собак
draw_cat - Отрисовка класса
info_cat - Информация о классе
remove_dogs_periodically - Атака собак
sell_cat - Продажа кота
handle_mouse_click_cat - Размещение кота

	CreateDog - Класс для создания собаки
create_dog - Создание класса
spawn_dog - Спавн собаки

	Dog - Собака на поле битвы
create_dog - Создание класса
move_dog - Движение собаки
info_dog - Отрисовка информации о собаке
bump_dog - Дамаг от котов
attack_dog - Обработка атаки собаки

	Game functions - Функции игры
main_menu - Функция главного экрана
menu_info - Функция с информацией о игре
lose_screen - Функция, показ экрана смерти, ваш скрипт выполнится
lose_event - Обработка ивентов в lose_screen (как event)
lose_ui - интерфейс lose_screen
info - Функция, отоброжение любой информации вводящей в аргументы, ваш скрипт выполнится
bonus - Функция, бафы и дебафы, ваш скрипт выполнится
bonus_loade - Загрузка бонусов, ваш скрипт выполнится
bonus_loade_event - Обработка ивентов в загрузке bonus
bonus_loade_ui - Интерфейс загрузки bonus
bonus_event - Обработка ивентов в bonus
bonus_ui - Интерфейс bonus
start - Функция, Прогрузка начальных переменных и тд, запуск игры, здесь нельзя выполнить ваш скрипт (он выполняется в скрипте main.py)
game - Функция, сама игра. (Обработка игровых данных), ваш скрипт выполниться но в качестве обаботки игровых данных

	Game
game_ui - Загрузка ваших ui (интерфейс элементов)
game_event - Загрузка ваших ивентов, к примеру "if event.type == pygame.KEYDOWN:", "if event.key == pygame.K_SPACE:"
game_game - Обработка игровых данных (Функция, сама игра.)
game_pause - Обработка паузы
game_die - Обработка смерти игрока
	Cmd
cmd - Свои консольные команды для игры

	Примеры:
В main.py добавляем строчку кода:
Script('Data/Mods/Main/Scripts/cat_supers.py', 'remove_dogs_periodically')
И весь скрипт на ульту котов пишем в этот файл путь которого мы указали, готово ! Вы добавили свой уникальный скрипт
